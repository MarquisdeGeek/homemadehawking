# homemadehawking
An Arduino-based speech synthesizer just like Stephen Hawking.

See it in action at : http://youtu.be/1eSkpLIrBUQ

## Components

* Arduino
* Shift register - 74HC595
* Narrator - SPO256
* Wires. Lots of wires.
